Hugoの絵文字

テーマの設定を変更することで利用できるようになります。（たぶん）

Configファイルで設定追加

設定ファイルの config.toml に下記を追加します。

enableEmoji = "true"

これだけです。

config.yml の場合はこちら。

enableEmoji: true

参考
Configure Hugo | Hugo

使い方

記事に ：smile： という感じのコードを入れるだけです。
展開しないように全角のコロンで書きましたが、実際は半角で書きますよ。

半角コロンを使って書くと → :smile:
